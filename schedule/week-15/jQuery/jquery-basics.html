JavaScript - jQuery

 

What is jQuery?

jQuery is an open source Javascript framework. Just like Bootstrap is a CSS framework. jQuery provides 'wrappers' or functions that simplify a lot of native JS functions.

 

Including jQuery

The line below includes the jQuery library hosted at Google CDN

                        
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>

You can also download it locally and include it but I would recommend otherwise.

 
 
 

Calling jQuery Functions

                        
document => jQuery => $
 

Example - Writing to the DOM



document.getElementById("emptyContainer") => jQuery("#emptyContainer") => $("#emptyContainer")
 
 
 
 

jQuery Selectors

DOM Selectors in jQuery work similar to CSS3 selectors


                        
$("p") - selects all HTML 'p' element(s)

                        
$("#emptyContainer") - selects HTML element whose's ID is emptyContainer

                        
$(".container") - selects HTML element(s) whose's CLASS is emptyContainer

All simple and advanced CSS3 selectors work similarly in jQuery. See jQuery Selectors in action.
