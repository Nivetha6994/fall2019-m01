What is DataTables?

It is a plugin that is built on top of the jQuery library. It provides advanced interaction controls to a HTML Table

How do we get started?

  1. Include the jQuery JS File
  2. Read the Documentation - https://datatables.net/examples/basic_init/zero_configuration.html
  3. Include the DataTable JS File
  4. Include the DataTable CSS File
  5. 'Bind' DataTables to the HTML Table
